3.flex语言开发中涉及的文件类型

  • .as 文件(相当于js)

    语法上风格,在java基础上融入了js 。

  • .mxml 文件 （相当于html）

  • .xml 文件 (参数配置文件)

  • .swf .swz 文件

    .mxml 编译后的文件，可以理解为 java .class 文件
